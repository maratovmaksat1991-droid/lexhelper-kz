# LexHelper KZ

Персональный AI-помощник казахстанского юриста.

Загрузите свои документы — система изучит ваш стиль и будет генерировать новые документы с учётом норм РК.

## Запуск локально

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Gemini API Key

Бесплатно: [aistudio.google.com](https://aistudio.google.com) → Get API Key → Create API Key.

Лимит бесплатного тарифа: 1500 запросов/день.

## Деплой на Streamlit Cloud

1. Залейте репо на GitHub
2. Зайдите на [share.streamlit.io](https://share.streamlit.io)
3. New app → выберите репо → `app.py` → Deploy
